Ramas

La rama master solo se toca para mergear y lo que esté en la master debe estar desplegado o debería, el código de otros entornos como test, desarrollo, etc debe estar en otras ramas.


git branch create a branch using git branch
- git checkout -b [rama a crear]
- git branch -b [nombre rama a crear] origin/[nombre rama a copiar] ==== creacion de ramas
- git branch -r (ver ramas remotas)

git checkout switch to a branch using git checkout NOTE: The -b flag creates the branch if it does not already exist.
git status check the status of files changed using git status
git log review recent commits using git log NOTE: --oneline para verlo todo en una linea
git revert revert changes using git revert
git branch get a list of branches and active branch using git branch
git merge merge changes in your active branch into another branch using git merge

Mover una rama


#mover
git branch -m [old-branch] [new-branch]

Cambiar el nombre de una rama


#1. moverse a la rama que queremos cambiar
git checkout [old-branch]
#2. seleccionar nuevo nombre
git branch -m [new-branch]
#3. sincronizar con el repositorio remoto
git push origin -u :[old-branch] [new-branch]
