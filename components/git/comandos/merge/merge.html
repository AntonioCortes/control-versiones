Unión de ramas

  • Para unir una rama con aquella en la que estás posicionada se usa git merge
  • Para abortar el merge se utiliza: git merge --abort

git merge [nombre de la rama]

git merge --abort

Comparación de archivos

Git permite comparar archivos entre ramas utilizando el comando git diff --name-status


# ver archivos diferentes entre 2 ramas
git diff --name-status [rama1][rama2]

Visualización de objetos

Para ver información sobre distintos elementos ya sean ramas, commits, etc se utiliza el comando git show


git show [objeto]
git show --name-status [objeto]
git show --name-only [objeto]
git show --pretty [objeto]

Unión evitando espacios en blanco

NOTA: Utilizar este comando al mergear cuando hay conflictos con los espacios en blanco


#Comando:
git merge -Xignore-all-space [nombre de rama]
